Cine-ku

Modern Movie Streaming Hub

Tentang Cine-ku

Cine-ku adalah website pencarian dan katalog film yang memanfaatkan data dari TMDb API. Pengguna dapat menjelajahi film populer, film dengan rating tertinggi, film yang akan datang, mencari film secara real-time, melihat detail film, trailer, serta menyimpan film favorit ke wishlist.

Pembuat

Nur Faiz Bekti Setiadi

Website ini dikembangkan menggunakan HTML, CSS, dan Vanilla JavaScript dengan integrasi TMDb API untuk data film, Firebase Authentication untuk sistem login, serta Firebase Hosting untuk deployment aplikasi.

Sumber Data

Seluruh data film berasal dari The Movie Database (TMDb).

Cine-ku tidak diafiliasi atau didukung oleh TMDb.

Fitur

  • Login Google (Firebase)
  • Banner Carousel
  • Slider film horizontal
  • Detail film + trailer YouTube
  • Realtime search
  • Wishlist (Local Storage)
